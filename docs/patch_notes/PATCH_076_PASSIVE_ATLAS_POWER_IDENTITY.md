# Patch 076 — Passive Atlas Power and Identity Rewrite

This patch upgrades the 900-node passive atlas data from bland generated filler into a stronger original ARPG passive baseline.

It preserves the 900-node scale while making the nodes more useful:

- stronger small passive values
- travel nodes that still carry useful power
- named notables, masteries, keystones, and bridge nodes
- large keystone upside/downside tradeoffs
- class-region identity for Sorceress, Huntress, and Warrior
- map / forge / gem progression nodes mixed into the atlas
- both readable Title Case stats and snake_case stat aliases for future systems
- compatibility functions for older and newer passive-panel/system code

This is still text/data-first. It is not the final visual passive tree UI.
