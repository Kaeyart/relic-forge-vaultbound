# Patch 010 — Physical Hub, No Menus

## Design correction

The hub is a physical place.

No command menu.
No giant hub overlay.
No full-screen forge menu.
No text-wall passive menu.

The player walks around and interacts with physical stations.

## Physical stations

### Contract Gates

Walk to a contract gate and press `E` to start that contract.

### Forge Anvils

Walk to a recipe anvil and press `E` to craft that recipe.

### Mastery Shrines

Walk to a shrine and press `E` to spend a mastery point in that branch.

### Skill Altars

Walk to a skill altar and press:

- `E` to upgrade that skill
- `X` to toggle it in/out of your loadout

### Stash / Armory

- Stash chest: `E` deposits backpack into stash
- Stash item pedestals: `E` withdraws that item
- Backpack item pedestals: `E` equips, `X` salvages
- Armory rack: `E` equips first backpack item, `X` salvages first backpack item

## Minimal UI only

The only UI should be:

- a small top status chip
- a small bottom prompt
- a small notice chip

No hub command screens.

## Controls

In hub:

- Walk around
- `E` interact
- `X` secondary interaction
- `F5` save

In dungeon:

- normal combat controls
- `Esc` returns to hub for now
