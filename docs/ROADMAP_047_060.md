# Roadmap 047–060

This roadmap is intentionally practical. It assumes the project is still a prototype but now has enough structure to be developed like a production slice.

## Patch 047 — Repo Hygiene + Production Baseline Cleanup

- Move patch notes and validators into archive folders.
- Ignore local backup clutter.
- Add current status and cleanup rules.
- Add consolidated validators.

## Patch 048 — Map Layout Quality Pass

- Make generated maps less rectangular and more readable.
- Add map room roles: start, pack room, side room, shrine room, boss arena.
- Add layout debugging display.
- Add collision/path checks.

## Patch 049 — Enemy Identity Pass

- Define enemy roles clearly.
- Add visible telegraphs.
- Add role-specific behavior: charger, archer, spitter, brute, summoner, boss.
- Avoid enemies feeling like dots.

## Patch 050 — Combat Feel Pass

- Improve skill feedback and impact.
- Improve enemy hit reactions.
- Improve damage numbers / floating feedback only if readable.
- Add camera/screen feedback carefully.

## Patch 051 — Observatory Accuracy Pass

- Fix useful/archetype/confusing/crafting-base classification.
- Add real gameplay telemetry events.
- Add better report summaries.

## Patch 052 — Map Device UX Pass

- Improve map stash and map selection.
- Show map tier, mods, boss, expected rewards, and danger.
- Add map-device scene placement in the hub.

## Patch 053 — Item Icon Placeholder Pack

- Add readable placeholder icons for weapon/armor/jewelry/relic/gems/maps/materials.
- No final art pressure; just visual clarity.

## Patch 054 — Crafting UX Pass

- Make forge actions clearer.
- Show forge potential, affix tiers, material costs, failure/lock states.
- Prevent accidental destructive actions.

## Patch 055 — Skill Gem UX Pass

- Improve uncut gem menus.
- Show support compatibility and reservation changes clearly.
- Show active skill behavior changes.

## Patch 056 — Passive Atlas MVP Pass

- Rebuild passive UI and data around fewer, stronger decisions.
- Avoid giant tree bloat before nodes are meaningful.

## Patch 057 — Hub Scene Production Pass

- Place map device physically.
- Make station arrangement readable.
- Add station labels/prompts and interaction zones.

## Patch 058 — First Art Production Batch: Inventory + Item Icons

- Focused item/UI icon batch only.
- No broad character/enemy art yet unless systems are stable.

## Patch 059 — First Art Production Batch: Enemies + Telegraphs

- Add readable enemy silhouettes and telegraph icons/effects.

## Patch 060 — Vertical Slice Review

- Run a proper playtest loop.
- Run Observatory reports.
- Decide whether to expand systems, polish combat, or produce art next.
