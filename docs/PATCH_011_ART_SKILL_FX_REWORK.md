# Patch 011 — Art + Skill FX Rework

## Goal

The game cannot keep feeling like debug dots.

Patch 011 adds a new presentation layer:

- player silhouette
- enemy role silhouettes
- biome floor treatment
- hub atmosphere
- skill-specific previews
- projectile trails
- impact bursts
- loot beams
- zone/trap readability
- hit pulses and light shake

## Important

This is not final sprite art. It is the missing visual language pass before real asset integration.

The game should become easier to read and more satisfying to test.

## Skill identities

### Fireball
Fire-colored projectile, warm trail, cast burst.

### Cleave
Melee arc preview, slash pulse, readable short-range attack shape.

### Frost Nova
Large circular radius preview, cold zone readability.

### Storm Lance
Fast bright projectile, lightning trail, thin line identity.

### Void Rift
Dark purple targeting zone, circular distortion, rift lines.

### Blade Trap
Diamond trap marker, armed hazard ring.

## Enemy identities

- Grunt: round hostile body
- Archer/Spitter/Acolyte: diamond caster/ranged silhouette
- Brute/Knight: heavy hex silhouette
- Hound: wedge/chaser silhouette
- Boss: larger aura and boss label

## What this replaces

Patch 011 disables the old Patch 005 readability overlay so the visual layers do not fight each other.

## Next art steps

- physical room props
