# Production Baseline

## Current project stage

Early vertical-slice production prototype.

This is not yet content production. It is still system stabilization and workflow stabilization.

## Core fantasy

Find rough items, sculpt them through deterministic-leaning crafting, discover broken skill/item interactions, and push map-like activities for better loot.

## Current design pillars

- Items are journeys, not stat sticks.
- Uniques should change skills or unlock archetypes.
- Skill gems and support gems create build identity.
- Crafting should be controlled and potential-limited.
- Combat must remain readable even with wild synergies.
- Beginners should understand obvious choices.
- Experts should find hidden leverage.

## Current technical pillars

- Scene-authored UI and hub layout.
- Modular scripts by system.
- Data-driven item/gem/map direction.
- Dev tools and Observatory reports for testing.
- Git checkpoints before risky work.

## What not to do right now

- Do not start broad final art production before systems stabilize.
- Do not add 200 affixes blindly.
- Do not replace manually authored UI scenes with generated layout scripts.
- Do not turn the project back into a single-script prototype.
- Do not tune from Observatory reports until classifier accuracy is fixed.
