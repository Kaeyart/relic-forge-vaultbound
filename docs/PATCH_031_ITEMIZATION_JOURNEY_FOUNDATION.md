# Patch 031 — Itemization Journey Foundation

This patch turns item drops into a more coherent ARPG itemization foundation.

## Adds

- Base item definitions.
- Normal / Magic / Rare / Unique rarity generation.
- Explicit prefixes and suffixes.
- Forge potential per item.
- Readable implicit, prefix, suffix, and unique-effect display.
- Build-changing unique flags.
- Better comparison text.
- Salvage rewards tied to rarity and affix tags.
- Design lock docs for itemization, combat/builds, and progression mastery.

## Philosophy

Items are rough gems. The player should find imperfect power and sculpt it.

Crafting should lean deterministic and control-oriented. Loot is the slot machine; crafting is the player's attempt to impose order.

Uniques should alter skills or create archetypes.
